StudyBuddy AI 伴学系统 — 部署教程
=====================================

写给电脑小白的部署指南，跟着步骤走就行。


一、你需要准备什么？
--------------------

 需要的             要求               没有的话怎么办？
 ─────────          ────               ────────────────
 一台电脑           Windows/Mac/Linux   —
 Python             3.11 或更新        去 python.org 下载安装
 这个项目文件夹     整个 AI伴学 文件夹  用 U 盘从有项目的电脑复制过来

※ 不需要 Git，不需要 Node.js（生产模式）。


二、三个主要文件（记牢它们）
----------------------------

 首次部署.py    →  新电脑上第一次运行。自动装环境、安装依赖、启动。
                   只用一次就够了。

 开启服务.py    →  以后每次双击它就能启动（生产模式）。
                   不用开命令行，跟打开游戏一样。

 关闭服务.py    →  双击它就能停止服务。

看见了吧？只需要记住这三个文件的名字就行了。
不懂哪个文件干什么用的时候，看「首次」「开启」「关闭」这三个词就够了。


三、第一次使用（新电脑）
------------------------

如果你第一次在这台电脑上运行，请按以下步骤：

 第 1 步：双击「首次部署.py」

 第 2 步：会弹出一个黑色窗口，按回车键开始。

 第 3 步：它会自动：
   √ 检查 Python 版本
   √ 创建虚拟环境
   √ 安装需要的依赖包
   √ 检查前端文件
   √ 启动服务并打开浏览器

 第 4 步：看到成功提示后，浏览器会自动打开。

 第 5 步：服务启动后不要关闭黑色窗口，关掉它服务就停了。
          以后每次用「开启服务.py」启动就行。


四、日常使用
------------

 启动：双击「开启服务.py」
       → 等几秒钟，浏览器会自动打开。
       → 以后每次用这个，不需要开命令行。

 停止：双击「关闭服务.py」
       → 看到提示"所有服务已停止"就行了。

 注意：关掉黑色窗口 = 服务停止。要随时用就让它开着。


五、出问题了怎么办？
--------------------

 问题 1：双击 py 文件没反应

   → 右键点文件 → 选「打开方式」→ 选 Python
   → 如果还是没有，可能是 Python 没装好

 问题 2：提示「Python 不是有效的命令」

   → 电脑没装 Python。
   → Windows：去 python.org 下载安装，安装时记得勾选
     「Add Python to PATH」
   → macOS：打开终端输入 brew install python
   → Linux：打开终端输入 sudo apt install python3

 问题 3：提示「未找到虚拟环境」

   → 运行「首次部署.py」就好了，它会自动创建。

 问题 4：端口被占用

   → 先双击「关闭服务.py」清理一下。
   → 如果还不行，重启电脑再试。

 问题 5：页面打开了但是空白

   → 前端文件没构建好。
   → 找一台有 Node.js 的电脑，双击「首次部署.py」，
     或者运行 python start.py --build
   → 然后把 frontend/dist 文件夹复制过来。

 问题 6：想看哪里出错了

   → 打开 backend 文件夹里的 server.log 文件
     （用记事本打开就行）


六、两种启动模式的区别
-----------------------

 生产模式（日常使用）   开发模式（给技术人员）
 ────────────────       ────────────────
 双击「开启服务.py」     双击「start.py」或终端运行
 不需要 Node.js          需要安装 Node.js
 启动快（几秒）          启动慢（等编译）
 1 个端口就够了          需要 2 个端口

 普通用户选生产模式就够了。


七、数据存在哪里？
------------------

你的题目、答案、设置都存在项目文件夹外面：

 桌面/
  ├── AI伴学/          ← 项目文件夹（代码）
  └── AI伴学数据/      ← 你的数据（题目、答案等）

数据不放在项目里面，所以更新代码不会丢数据。
建议定期把「AI伴学数据」文件夹备份到 U 盘。


八、换到新电脑（完整步骤）
--------------------------

 第 1 步：在旧电脑上

   → 双击「首次部署.py」或者运行 python start.py --build
   → 把整个「AI伴学」文件夹复制到 U 盘

 第 2 步：复制到新电脑

   → 把 U 盘里的「AI伴学」文件夹复制到新电脑桌面

 第 3 步：在新电脑上

   → 双击「首次部署.py」等待完成
   → 以后每次双击「开启服务.py」启动


九、用 AI 帮你部署（进阶）
--------------------------

如果你电脑上装了 OpenCode（好比一个聪明的机器人助手），
可以让它帮你完成部署：

 第 1 步：安装 OpenCode
   → 双击 tools 文件夹里的 install_opencode.py

 第 2 步：启动 OpenCode
   → 在命令行输入 opencode 回车

 第 3 步：让它帮你干活
   → 输入 /setup  机器人就会检查你的电脑环境
   → 输入 /deploy 机器人就会自动部署启动


十、常见问题
------------

 问：我不小心关了黑色窗口，服务还在吗？
 答：关了窗口服务就停了。重新双击「开启服务.py」就好。

 问：怎么让电脑开机自动启动？
 答：目前需要手动双击启动。以后可以帮你做一个开机自启。

 问：别人能用手机访问吗？
 答：可以。在同一个 WiFi 下，手机浏览器输入网址即可。
     启动时会显示类似 http://192.168.x.x:xxxx/ 的地址。

 问：数据会不会丢？
 答：数据存在「AI伴学数据」文件夹里，代码坏了数据不会丢。
     建议定期把这个文件夹复制到 U 盘备份。
